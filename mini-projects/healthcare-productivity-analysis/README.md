\# Healthcare Productivity \& SLA Analysis



\## Objective

Analyze healthcare coding productivity data to evaluate:

\- Coder performance

\- Turnaround Time (TAT)

\- SLA adherence



\## Dataset (Simulated)

\- coder\_name

\- chart\_id

\- tat\_days

\- status



\## Tools Used

\- Python

\- Pandas

\- Matplotlib



\## Key Insights

\- Identified top-performing coders

\- Flagged charts breaching SLA

\- Categorized performance using business rules



\## SQL Analysis

The same business logic was implemented using SQL:

\- Aggregation using GROUP BY

\- Performance categorization using CASE WHEN

\- SLA breach identification using WHERE conditions



This demonstrates ability to translate business rules between SQL and Python.
