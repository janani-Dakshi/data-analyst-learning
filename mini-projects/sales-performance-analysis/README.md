\# Sales Performance Analysis



\## Objective

Analyze sales data using Python and Pandas to identify:

\- Best performing regions

\- Monthly sales trends

\- Distribution of sales amounts



\## Tools Used

\- Python

\- Pandas

\- Matplotlib



\## Key Insights

\- South region recorded the highest sales

\- Sales show an upward trend over months

\- Most sales amounts fall within a mid-range
